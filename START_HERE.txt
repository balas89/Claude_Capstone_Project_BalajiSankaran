🚀 QUICK START (5 MINUTES):

  1. Set your API key:
     export ANTHROPIC_API_KEY="sk-ant-..."

  2. Install dependencies:
     python3 -m pip install --user fastapi uvicorn anthropic pydantic python-dotenv streamlit requests langchain

  3. Start all services (6 terminals):
     Terminal 1: python3 -m mcp_servers.applicant_db
     Terminal 2: python3 -m mcp_servers.risk_rules_db
     Terminal 3: python3 -m mcp_servers.decision_synthesis
     Terminal 4: python3 -m mcp_servers.notification_system
     Terminal 5: python3 -m microservices.app
     Terminal 6: streamlit run ui/streamlit_app.py

  4. Open UI:
     http://localhost:8501
