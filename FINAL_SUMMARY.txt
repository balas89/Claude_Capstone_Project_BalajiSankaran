🎨 WHAT THIS COMMAND DOES
───────────────────────────────────────────────────────────────────────────

1. Builds Docker image with all services
2. Creates Docker network (loan-approval-network)
3. Starts 6 containerized services in order:
   ✓ Applicant DB MCP (8001)
   ✓ Risk Rules DB MCP (8002)
   ✓ Decision Synthesis MCP (8003)
   ✓ Notification System MCP (8004)
   ✓ FastAPI Microservice (8000)
   ✓ Streamlit UI (8501)
4. Sets up inter-service networking
5. Enables health monitoring
6. System ready in ~30-40 seconds
