 HOW TO RUN THE PROJECT
═══════════════════════════════════════════════════════════════════════════════

PREREQUISITES (Already Done):
  ✓ Virtual environment (venv)
  ✓ Dependencies installed (requirements.txt)
  ✓ Configuration (.env file)
  ✓ API key set (ANTHROPIC_API_KEY)

OPTION 1: Simple Startup (Recommended)
  $ bash run_all.sh
  
  This will:
  - Start all 6 services
  - Create logs/ directory
  - Auto-open http://localhost:8501
  - Ready in 3-5 seconds

OPTION 2: Docker (Single Command)
  $ docker-compose up --build
  
  Then visit: http://localhost:8501

OPTION 3: Manual Startup (Advanced)
  Terminal 1: python -m uvicorn mcp_servers.applicant_db:app --port 8001
  Terminal 2: python -m uvicorn mcp_servers.risk_rules_db:app --port 8002
  Terminal 3: python -m uvicorn mcp_servers.decision_synthesis:app --port 8003
  Terminal 4: python -m uvicorn mcp_servers.notification_system:app --port 8004
  Terminal 5: python -m uvicorn microservices.app:app --port 8000
  Terminal 6: streamlit run ui/streamlit_app.py --server.port 8501
