📁 PROJECT STRUCTURE SUMMARY
═══════════════════════════════════════════════════════════════════════════════

🤖 AGENTS LAYER (agents/)
────────────────────────────────────────────────────────────────────────────────
  applicant_agent.py              Agent 1: Analyzes applicant profile & data
                                  - Validates completeness, calculates income stability
                                  - Assesses employment risk, summarizes credit history
                                  
  financial_risk_agent.py         Agent 2: Evaluates financial metrics
                                  - Calculates DTI, LTI ratios
                                  - Detects financial anomalies
                                  
  loan_decision_agent.py          Agent 3: Synthesizes final decision
                                  - Applies business rules engine
                                  - Makes Approve/Reject/Review decision
                                  
  compliance_agent.py             Agent 4: Ensures compliance & notifications
                                  - Sends notifications, records audit trail
                                  - Logs decisions for regulatory requirements


🔌 MCP SERVERS (mcp_servers/) - 4 Independent Microservices
────────────────────────────────────────────────────────────────────────────────
  applicant_db.py                 MCP Server 1 (Port 8001)
                                  - Applicant data queries
                                  - Credit history lookups
                                  
  risk_rules_db.py                MCP Server 2 (Port 8002)
                                  - Financial risk calculations
                                  - DTI/LTI computations
                                  
  decision_synthesis.py           MCP Server 3 (Port 8003)
                                  - Business rules engine
                                  - Decision logic (NOT LLM-based)
                                  
  notification_system.py          MCP Server 4 (Port 8004)
                                  - Notifications & audit logging
                                  - Case tracking

🔄 EXECUTION FLOW
════════════════════════════════════════════════════════════════════════════════

1. USER INTERACTION
   └─> Streamlit UI (streamlit_app.py)
       • User fills 9-field form
       • Clicks "Submit Application"

2. DATA SUBMISSION
   └─> FastAPI endpoint (microservices/routes.py)
       • HTTP POST /apply-loan
       • Pydantic validates (microservices/schemas.py)

3. WORKFLOW ORCHESTRATION
   └─> LangGraph (orchestration/workflow.py)
       • Initialize state
       • Agent 1 & 2 run in PARALLEL (both simultaneously)
       • Agent 3 runs after parallel
       • Agent 4 runs after Agent 3
       • Finalize

4. AGENT EXECUTION
   ├─> Agent 1 (applicant_agent.py)
   │   └─> MCP 1 (mcp_servers/applicant_db.py)
   │       └─> Mock data (utils/mock_data.py)
   │
   ├─> Agent 2 (financial_risk_agent.py)
   │   └─> MCP 2 (mcp_servers/risk_rules_db.py)
   │       └─> Mock data (utils/mock_data.py)
   │
   ├─> Agent 3 (loan_decision_agent.py)
   │   └─> MCP 3 (mcp_servers/decision_synthesis.py)
   │       └─> Business rules (utils/decision_rules.py)
   │
   └─> Agent 4 (compliance_agent.py)
       └─> MCP 4 (mcp_servers/notification_system.py)

5. RESPONSE BUILDING
   └─> LoanDecisionResponse (microservices/schemas.py)
       • Decision, risk score, factors
       • Detailed analysis
       • Audit trail

6. UI DISPLAY
   └─> Streamlit (streamlit_app.py)
       • Color-coded decision
       • Risk score & confidence
       • Detailed analysis tabs
       • Application history

📊 DECISION LOGIC SUMMARY
════════════════════════════════════════════════════════════════════════════════

Hard Rejections (Automatic ❌)
  • DTI ≥ 50%
  • Credit Score < 600
  • 2+ severe anomalies (bankruptcy, foreclosure)

Approvals (All criteria met ✅)
  • DTI < 43%
  • Credit Score ≥ 650
  • LTI < 3.0x
  • Low employment risk
  • No major anomalies

Manual Review (Mixed signals 🔄)
  • DTI 43-50%
  • Credit Score 600-650
  • Multiple moderate factors

Risk Score Calculation:
  • DTI (30% weight): 0-0.30
  • Credit (25% weight): 0-0.25
  • LTI (20% weight): 0-0.20
  • Income Stability (15% weight): 0-0.15
  • Employment Risk (10% weight): 0-0.10
  • Anomalies (5% weight): 0-0.10
  Total: 0.0 (safest) to 1.0 (riskiest)
